To activate the AI Trading Analyst, please configure at least one LLM provider in your backend .env file:

# 1. Local LLM via Ollama (Recommended & Free)
OLLAMA_BASE_URL=http://localhost:11434/v1
OLLAMA_MODEL_NAME=qwen2.5-coder:7b

# 2. Cloud LLM via Google Gemini
GEMINI_API_KEY=your_key_here
GEMINI_MODEL_NAME=gemini-1.5-flash

# 3. Cloud LLM via OpenAI GPT
OPENAI_API_KEY=your_key_here
OPENAI_MODEL_NAME=gpt-4o-mini

Restart the backend server after editing your .env file.
